Fuzzy Logic Tab

1 - HTTP Response Result Score
2 - Ignore Fuzzy Logic trigger (all results are populated)
3 - Filtering minimum / maximum values
4 - Results equal to minimum value
5 - Results not equal to minimum value
6 - All results
7 - Results outside minimum / maximum value rane
8 - Results inside minimum / maximum value rane
9 - Update the result list

The Fuzzy Logic tab is used to control the display of the results.

J-Baah compares the HTTP responses from the Brute Force attack with a base-line response (the Base Response) obtained prior to the attack being executed.

Scores are float-based, with a score of 1 being a perfect match with the base response (ie: The HTTP response and the base response are identical).

Whilst the brute force attack is being executed, the results are only populated in the result list if the score of the results matches the currently selected criteria, unless the check box "Ignore Fuzzy Logic Trigger" is checked.

Context Menu
Two commonly used features "All but these", and "Only these" are available in the result list's context menu. Selecting either of these options whilst a row is currently highlighted will set the Fuzzy Logic filter for you and update the results.
