Actions Tab

The actions tab is used to control the brute force attack. Not only does it provide control of the number of scanning threads used in the attack, but it also provides options for "Base Response", "Start", "Stop", "Pause" and "Resume", although not all of these features will always be available or enabled.

Base Response
Before an attack can be initiated, a base response should be obtained. This base HTTP response is used by J-Baah for comparison.

Start
Once HTTP requests, parameters and targets have been specified and a base response obtained, this button is used to start a brute force attack.

Note: This destroys all results from prior runs. If results from previous scans have been obtained and are worth saving, please export them first.

Stop
This option is only enabled when scans are actually running. It stops the brute force attack.

Pause / Resume
This option is only enabled when scans are actually running. It used used to pause or resume the brute force attack.
